## 图片要求

- 所有图片应为高质量JPG或PNG格式
- 建议使用白色或简洁背景，突出产品细节
- 图片应保持一致的风格和色调
- 为保证网站加载速度，图片应适当压缩

## 注意事项

在没有实际图片的情况下，网站会使用placeholder.com的占位图片服务。这些占位图只用于开发和测试目的，不应用于生产环境。
